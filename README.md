# Smart Reader 📚

Smart Reader is a Django-based web application that helps users manage and read articles efficiently.

## Features

* User authentication (Login / Signup)
* Article reading interface
* Admin dashboard
* Reading list management
* Privacy flow for user data

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript
* SQLite

## Installation

1. Clone the repository
   git clone https://github.com/Sanjajgiri/Smart-Reader.git

2. Navigate to the project directory
   cd Smart-Reader

3. Install dependencies
   pip install -r requirements.txt

4. Run the development server
   python manage.py runserver

5. Open in browser
   http://127.0.0.1:8000/
